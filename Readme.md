This project serves for the individualization of a modified csl for printing Chinese references in different formats. The csl is to be used with Zotero. The project creates a simple website in which the desired modifications can be selected and a modified csl is created, either for download or for copy-and-paste. The created csl file is a modification of the CMOS17 (shortened notes and bibliography) created by Andrew Dunning under the Creative Commons Attribution-ShareAlike 3.0 License.
